PURPOSE

transformIndSmaller Transforms indices from bigger space to smaller space

SYNOPSIS

function outputIndex = transformIndSmaller(index,transformMat)

DESCRIPTION 

transformIndSmaller Transforms indices from bigger space to smaller space
   index = the indices in bigger space you are shaping into smaller space
   transformMat = logical array the size of bigger space specifying which
   indices are part of the smaller space.

   Assumes that order is retained between bigger space and smaller space
