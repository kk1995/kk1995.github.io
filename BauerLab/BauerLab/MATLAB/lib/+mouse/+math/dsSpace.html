dsSpace

PURPOSE

dsSpace Downsample 2D matrix spatially.

SYNOPSIS

function dataDS = dsSpace(data,factor,average)

DESCRIPTION 

dsSpace Downsample 2D matrix spatially.
   data = 2D matrix
   factor = 1 integer. The dimensions of data should better be divisible by
   the factor.
   average = if true, the multiple values for one pixel is averaged. If
   false, then only the value in the middle is considered.
