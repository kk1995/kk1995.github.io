getTriangleInd

PURPOSE

getTriangleInd Get indices of lower and upper triangle

SYNOPSIS

function [lower,upper] = getTriangleInd(n)

DESCRIPTION 

getTriangleInd Get indices of lower and upper triangle
   assumes diagonal to go from (1,1) to (n,n)
   Input:
       n = length of matrix
   Output:
       lower
       upper
   output order is done such that lower(x) is symmetric to upper(x)
