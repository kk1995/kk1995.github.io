rmvHbAbsLen

PURPOSE

rmvHbAbs Removes absorption by hemoglobin (cm*M)

SYNOPSIS

function [fluorDataHbAbsRemoved, fraction] = rmvHbAbsLen(fluorData,hbOData,hbRData,hbOAbsCoeff,hbRAbsCoeff)

DESCRIPTION 

rmvHbAbs Removes absorption by hemoglobin (cm*M)
   Inputs:
       fluorData = 1D fluorescence data
       hbOData = 1D HbO change data (cm*M)
       hbRData = 1D HbR change data (cm*M)
       hbOAbsCoeff = 1x2 HbO absorption coefficient for two wavelengths
       hbRAbsCoeff = 1x2 HbR absorption coefficient for two wavelengths
   Outputs:
       fluorDataHbAbsRemoved = 1D fluorescence data with Hb effect removed
       fraction = ratio of fluorescence with Hb removed compared to
       original
